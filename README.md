# UCL 2018 Final: Substitution Impact Analysis

## Overview

This project analyzes how substitutions affected attacking patterns in the 2018 UEFA Champions League Final between Real Madrid and Liverpool.

The focus is on identifying structural changes in ball progression and shot creation before and after key substitutions, particularly Gareth Bale's introduction.

## Research Question

How did substitutions (Salah 29', Bale 60') impact team attacking structures and shot generation?

## Data

* Source: StatsBomb Open Data
* Match ID: 18245

## Key Analyses

* Pass network visualization (16-zone pitch control)
* Team shot maps
* Gareth Bale individual shot map
* Substitution information
* Heatmap comparison (before vs after substitutions)

## Key Findings

* Real Madrid's attacking density increased centrally after Bale's introduction
* Shot quality improved post-substitution
* Liverpool's right-side progression dropped significantly after Salah's injury
